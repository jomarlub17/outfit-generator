# Outfit Generator 

A smart wardrobe management app that helps you organize your clothes and generate random outfit combinations.

**Live Demo:** Coming Soon!

## Features

- Add, view, and delete clothing items
- Generate random outfit combinations
- Data persists in browser localStorage
- Responsive design for mobile and desktop
- Modern, clean UI

## Tech Stack

- **Frontend:** React, JavaScript, CSS
- **State Management:** React Hooks (useState, useEffect)
- **Data Persistence:** Browser localStorage
- **Deployment:** To be deployed soon with Vercel

## How It Works

1. Add clothing items to your wardrobe (type, color, material, season)
2. Click "Generate Outfit" to get a random combination
3. Your wardrobe data is automatically saved to your browser
